# Note Icon Placeholders

Drop your preferred PNG notation glyphs in this folder using the following filenames so the palette can load them without cross-origin issues:

- whole.png
- half.png
- dotted-half.png
- quarter.png
- dotted-quarter.png
- eighth.png
- dotted-eighth.png
- triplet-eighth.png
- sixteenth.png
